Re:charge Web 🌱

● 전기차 충전소 정보 기반 웹 서비스 (2차 팀 프로젝트)

● 프로젝트 개요

Re:charge Web은
전기차 이용자를 위한 충전소 정보 제공 웹 서비스로,
사용자 인증과 CRUD 기능을 중심으로 프론트엔드–백엔드 연동 구조를 학습·구현한 프로젝트입니다.

React 기반의 SPA 구조에서
Spring Boot REST API와 Oracle DB를 연동하여
실제 서비스 흐름에 가까운 웹 애플리케이션을 구현하는 것을 목표로 했습니다.
